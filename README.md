## Project Motivation<a name="motivation"></a>

This is Udacity's Data Scientist Nanodegree project. This project use CRISP-DM Process to analyze the data related to AirBnB data for housing in Seattle city. Using this data we will try to find answers to some of the interesting question that can give us some insight in a business point of view.

1. What features are most important that determine the price ?
2. Where are the most requested neighbourhoods in Seattle ?
3. What month does tourist visits Seattle most ? 
4. What is the best way to predict price for a certain period ?


## File Descriptions <a name="files"></a>

The following are the files used in the project. 
1. SeattleAirBnBDataAnalysis.ipynb: This files contains the steps followed and code used to find answers to above question. Details of individual step taken are in Markdown cells for better understanding.
2. README.md: This gives project overview. 
3. Data files: 
	"Calender.csv" contains listing  availablilty and price on a specific date.
	"listings.csv" contains the details about the property such as location, number of rooms, price(monthly or weekly) e.t.c.
	"review.csv" contains the listing reviews.
